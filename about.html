About our project

Our Project is Regarding automatic payroll .As we are designing a website for it we should have a theme to proceed so we created a company named tradinglaze and we are working to create a automatic payroll for the company tradinglaza.

Some of the features of our project are:
  • Logins for employees ,Admin,and HR.

  • Admin can add new HR and HR can add new employees.

  • Salaries of the employees is monitored.

  • Employees work location is monitored.

  • HR and Admin can monitor employees work locations.

  • While logging in picture of the employee is taken.

  • HR and Admin can check whether the employee logged in or not by confirming photo.

  • While logging out picture of the employee is taken.
